About my name

 

In Chinese, the family name is in the front, while the given name comes after.

 

The single character '翊' is my given name, which means 'to assist, to help' or 'the wing' (see Wiktionary). This character is not very common in modern Chinese, which makes it so unique as well as beautiful. That's why I like it so much :）.

 

In China, people usually call me the full name "Dai Yi" in a Chinese way. However, if you are English (or other languages e.g. German, Danish and Telugu) speaker, I would prefer if you just call me 'Yi'.
